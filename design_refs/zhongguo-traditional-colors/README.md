# Zhongguo Traditional Colors Local Library

Local design reference data generated from <https://colors.xiaoxiaodong.ai/>.

## Files

- `build_library.mjs`: Fetches the source website data and rebuilds the local JSON files.
- `metadata.json`: Source URLs, generation time, count, and license note.
- `colors.json`: 742 normalized colors with `id`, Chinese `name`, `hex`, `rgb`, `cmyk`, `pinyin`, and image URL.
- `harmonies.json`: Color relationship map generated by the source project.
- `harmony_usage.json`: Human-readable guidance for each harmony relationship type.
- `booking_screening_tokens.json`: First UI token set for the booking form screening workflow.

## Rebuild

```bash
node design_refs/zhongguo-traditional-colors/build_library.mjs
```

## Current Booking Screening Direction

The first token set avoids the default blue admin palette and uses restrained traditional colors:

- Background: `荼蘼白`, `茉莉白`, `奶油白`
- Text: `松烟墨`, `墨黑`
- Primary action: `青黛`
- Neutral support: `黛色`, `雾霾蓝`, `云峰灰`, `高级灰`
- Error: `茜裙`, `胭脂泪`
- Warning: `鎏金`
- Correction: `浅豆绿`, `翠青`
